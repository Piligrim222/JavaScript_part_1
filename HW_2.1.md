Задание 1.
----------
*Почему код дает именно такие результаты?*

`var a = 1, b = 1, c, d;`
`c = ++a; alert(c);           // 2`
`d = b++; alert(d);           // 1`
`c = (2+ ++a); alert(c);      // 5`
`d = (2+ b++); alert(d);      // 4`
`alert(a);                    // 3`
`alert(b);                    // 3`

------------
#### Решение:
**1.  c = ++a**
Cначала к переменной **а** добавили 1, потом переменной **c** присвоили значение переменной **а**;
`a = 1 + 1 = 2 -> c = a = 2`

**2. d = b++**
Cначала поместили в переменную **d** значение переменной **b** , а потом к переменной **b** добавили 1;
`d = b = 1 -> b = b + 1 = 2`

**3. c = (2+ ++a)**
`a = 2 + 1 = 3 -> c = 2 + 3 = 5`

**4. d = (2+ b++)**
`d = 2 + b = 2 + 2 = 4 -> b = 2 + 1 = 3`

**5.**
Из пункта 3: `а = 3`
**6.**
Из пункта 4: `b = 3`
